## Ponders
Ponders are coming soon!

Repackaged is a create addon that allows to transfer fluids and energy through packages.
This mod relies completely on Deployer, an API specialized on create addons. Our main aim is to bring stability and compatibility to these package networks, since other mods tend to have problems on this.

## Package Shelf
The package shelf is a new repackager-like block. It detects different packages from the same order (items and fluids) and puts all of them in the shelf if they are present.
This is our alternative to composite-packaging since it doesn't really fit in the game aesthetics and with this you can ensure correct order of packages
